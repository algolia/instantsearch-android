## How to run this example

### 1. Clone this repository

```sh
git clone git@github.com:algolia/instantsearch-android.git
```

### 2. Build the project

```sh
./gradlew :examples:android:assembleDebug
```

### 3. Install the app

```sh
./gradlew :examples:android:installDebug
```
