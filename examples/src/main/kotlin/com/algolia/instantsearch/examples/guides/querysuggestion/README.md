# Places for InstantSearch Android

Query Suggestions is a search experience that displays a list of possible queries that your users can select from as 
they type.

<img src="/docs/img/guide/suggestions.gif" width="300"/>

## How to run this example

### 1. Clone this repository

```sh
git clone git@github.com:algolia/instantsearch-android.git
```

### 2. Build the project

```sh
./gradlew :examples:assembleDebug
```

### 3. Install the app

```sh
./gradlew :examples:installDebug
```

### 4. Launch the example

The example is under `Guides > Query Suggestion`.
